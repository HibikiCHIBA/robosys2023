# Robosys2023

## Outline
千葉工業大学未来ロボティクス学科 上田先生の講義 ロボットシステム学 2023年度版の内容に沿って作成した課題の提出用リポジトリ. 

## Environment
* Ubuntu 20.04.4 LTS
* python 3.8.10

## Plus
Pythonプログラム. 標準入力までの整数の和を計算して, その結果を出力する. 例外の際は浮動小数点数が加算される.
