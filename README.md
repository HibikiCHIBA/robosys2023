# Robosys2023
千葉工業大学 上田隆一先生の講義 ロボットシステム学 2023年度版の内容に沿って作成した課題の提出用リポジトリである. 

## Environment
* Ubuntu 20.04.4 LTS
* python 3.8.10

(上記環境にてテスト済み)

## Plus
標準入力までの整数の和を計算して, その結果を出力する. 例外処理の際は浮動小数点数が加算される. 

#### Demo
```
$ seq 5 | ./plus  
15
```
